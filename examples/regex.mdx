---
title: Regular Expressions
description: Transform RegExp literals and methods using luau-regexp
---

This example shows how rbx-tsx transforms regular expression literals and methods to use the Roblox luau-regexp library.

<Tabs>
  <Tab title="Input (TypeScript)">
    ```typescript
    /**
     * Regular expressions — compiles to luau-regexp (RegExp)
     * Requires: ReplicatedStorage.Packages.RegExp (Roblox/luau-regexp)
     */
    
    // Regex literal → RegExp(pattern, flags)
    const digitPattern = /\d+/g;
    const emailPattern = /^[a-z]+@[a-z]+\.[a-z]+$/i;
    
    // .test(str) → regex:test(str)
    const hasDigits = digitPattern.test("abc123");
    const looksLikeEmail = emailPattern.test("user@example.com");
    
    // .exec(str) → regex:exec(str)
    const match = digitPattern.exec("price: 99");
    if (match) {
      // luau-regexp returns 1-based: match[1] = full match, match.index = position
      console.log(match[1], match.index);
    }
    
    // str.match(regex) → regex:exec(str)
    const parts = "hello world".match(/l+/g);
    
    // new RegExp(pattern, flags)
    const dynamicPattern = new RegExp("\\d{2,4}", "g");
    
    // str.replace(regex, repl) — uses regex.source with string.gsub (Lua pattern)
    // Complex regex patterns may need manual conversion
    const cleaned = "  spaces  ".replace(/\s+/g, " ");
    ```
  </Tab>
  <Tab title="Output (Luau)">
    ```lua
    -- Auto-generated by rbx-tsx
    -- Source: regex.ts
    
    local RegExp = require(game:GetService("ReplicatedStorage").Packages.RegExp)
    -- 
    --[[
        Regular expressions — compiles to luau-regexp (RegExp)
        Requires: ReplicatedStorage.Packages.RegExp (Roblox/luau-regexp)
    ]]
    -- Regex literal → RegExp(pattern, flags)
    local digitPattern = RegExp("\\d+", "g")
    local emailPattern = RegExp("^[a-z]+@[a-z]+\\.[a-z]+$", "i")
    -- .test(str) → regex:test(str)
    local hasDigits = digitPattern:test("abc123")
    local looksLikeEmail = emailPattern:test("user@example.com")
    -- .exec(str) → regex:exec(str)
    local match = digitPattern:exec("price: 99")
    if match then
        -- luau-regexp returns 1-based: match[1] = full match, match.index = position
        print(match[2], match.index)
    end
    -- str.match(regex) → regex:exec(str)
    local parts = RegExp("l+", "g"):exec("hello world")
    -- new RegExp(pattern, flags)
    local dynamicPattern = RegExp("\\d{2,4}", "g")
    -- str.replace(regex, repl) — uses regex.source with string.gsub (Lua pattern)
    -- Complex regex patterns may need manual conversion
    local cleaned = string.gsub("  spaces  ", RegExp("\\s+", "g").source, " ", 1)
    ```
  </Tab>
</Tabs>

<Note>
  Key transformations:
  - RegExp literals `/pattern/flags` become `RegExp("pattern", "flags")` constructor calls
  - `.test()` and `.exec()` methods are called with `:` syntax
  - `str.match(regex)` becomes `regex:exec(str)`
  - `str.replace(regex, repl)` becomes `string.gsub(str, regex.source, repl, 1)`
  - Array indexing in match results adjusts: TypeScript's `match[0]` (full match) becomes Luau's `match[2]` due to 1-based indexing with capture groups
</Note>
